###  使用说明:

1. 将项目导入Eclipse,将项目包中的product.sql文件导入数据库中
2. 配置资源文件中的properties中的数据库信息,然后将项目的webapps文件目录配置在TOMCAT的conf下面的SERVER.xml文件中
3. 启动TOMCAT,一切OK了.
